## 使用建议

1. **新手入门**：先阅读 [核心知识精华](./核心知识精华.md)
2. **深入学习**：进入各主题文件夹阅读详细内容
3. **查找灵感**：浏览 [Lenny播客精华](./Lenny播客精华/) 按主题查找
4. **原始搜索**：在 `原始数据/` 中进行全文搜索

## 与产品判官集成

本知识库是"产品判官"项目的知识来源。运行以下命令导入知识库：

```bash
cd ../backend
python ingest.py
```

这会将知识库内容向量化，供 AI 检索使用。
