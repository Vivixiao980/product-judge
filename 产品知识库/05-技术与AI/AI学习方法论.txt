                    AI学习方法论 - 核心观点提炼
                    来源：张咋啦（Zara）小红书笔记
================================================================================

【一、学习心态与原则】

1. 看一手信息，不看二手信息
   - Follow builders, not influencers
   - 最前沿的AI知识都是免费的，散落在X和YouTube上
   - 找对信息源，中国和硅谷没有信息差

2. Learn in public（公开学习）
   - 如果别人不知道你学了，就跟没有学一样
   - 学习心得公开分享，能吸引同频的人
   - 面向6个月前的自己，写"仰望15度角"的内容

3. Build first, learn later（先做后学）
   - AI时代的学习要倒过来：不是先学后做，而是先做后学
   - 学AI最好的方式：做个side project
   - 动手用产品，相信自己作为用户的体感

4. 降低心理门槛比时间管理更重要
   - 与其想怎么做时间管理，不如想想如何降低心理门槛、提升动力、做减法
   - 把学习变成爱好，而不是靠意志力坚持


【二、学习方法】

1. 完整看长视频/播客
   - 播客是最高质量的信息源：嘉宾水平高、一手信息、内容够深
   - YouTube是全世界最好的AI学习平台
   - 长视频是深度学习的最好方法之一

2. 主动消费，而非被动消费
   - 筛选出少数几个高质量信息源
   - 不刷信息流产品（包括twitter、短视频、朋友圈等）
   - 主动选择高质量信息源，而不是被动靠算法推荐

3. 用AI辅助学习
   - 用Claude/ChatGPT做随身老师
   - 把不懂的技术名词丢给AI解释
   - 用NotebookLM把内容转成AI播客讲解

4. 提升消费内容的仪式感，降低创作内容的仪式感
   - 消费内容要专注、深入
   - 创作内容要轻量、随手

【四、学习内容分类】

文科生学AI的重点：
1. 不是学代码/看论文
2. 而是：看YouTube、查Perplexity、上手用产品
3. 关注应用层创新，而非底层技术
4. 理解产品如何解决用户问题

需要关注的方向：
- AI产品的交互创新
- 应用层的商业模式
- 用户体验和产品设计
- AI时代的工作方式变化


【五、避免的误区】

1. 学AI ≠ 转码
   - AI时代，technical的定义在发生变化
   - 不需要从零学编程成为初级程序员
   - 而是要学会用AI工具实现想法

2. 不要被满屏代码吓到
   - 代码只是达到目标的手段
   - 重要的是知道要做什么，AI可以帮你实现

3. 不要追求面面俱到
   - 先想清楚学习目标，再去倒推学习方法
   - 不学什么跟学什么同样重要
